Une application de géolocalisation participative

Création d'une application de géolocalisation participative pour l'association Culture & Territoire ruraux, dans les Cévennes. Le but de ce projet était d'aider la répertorisation de différents objets du patrimoine cévennol, éparpillé à travers tout le territoire de la haute-vallée de l'Hérault. Ainsi, les touristes comme les riverains peuvent télécharger l'application, prendre des photos de différents points d'intérêt, et les envoyer sur la base de données de l'association.
